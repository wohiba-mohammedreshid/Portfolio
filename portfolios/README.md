# Professional Portfolio

This repository contains the files for a personal professional portfolio website. It includes:

- `index.html` - main page of the portfolio
- `assts/css/styles.css` - stylesheet for the site
- `assts/js/script.js` - JavaScript for interactive features
- `assts/images/` - folder for portfolio images and assets

## Setup

To view the portfolio, open `index.html` in a web browser. No build steps are required.
